# aminokaa
😎Замена слов в тексте😎
Эта программа позволяет заменить одно слово на другое в введённом тексте.
Пользователь вводит текст, затем указывает слово, которое нужно заменить, и новое слово. После этого программа выводит изменённый текст или сообщает, что такого слова нет.

🧐Как запустить🧐
Установите Python (если он ещё не установлен)
Скачайте файл main.py
Откройте терминал в папке с файлом
Запустите программу командой:
python main.py

😇Как пользоваться😇
Введите текст
Введите слово, которое хотите заменить
Введите новое слово
Получите результат на экране

👅Пример работы👅
Ввод:
Введите текст: я люблю программирование
Какое слово заменить: программирование
На что заменить: Python
Вывод:
Готово! Вот результат:
я люблю Python
