# aminokaa  
😎Замена слов в строке😎   
Эта программа позволяет заменить одно слово на другое в введённой строке.  
Пользователь вводит строку, затем указывает слово, которое нужно заменить, и новое слово. После этого программа выводит изменённую строку или сообщает, что такого слова нет.
 
🧐Как запустить🧐  
Установите Python (если он ещё не установлен)  
Скачайте файл main.py  
Откройте терминал в папке с файлом  
Запустите программу командой:  
python main.py  
  
😇Как пользоваться😇  
Введите строку  
Введите слово, которое хотите заменить  
Введите новое слово  
Получите результат на экране  
  
👅Пример работы👅  
Ввод:  
Введите строку: я люблю программирование  
Какое слово заменить: программирование  
На что заменить: Python  
Вывод:  
Готово! Вот результат:  
я люблю Python
