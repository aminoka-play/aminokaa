# aminokaa  
😎Замена слов в строке😎   

Эта программа позволяет заменить одно слово на другое в введённой строке.
Пользователь вводит строку, затем слово для замены и новое слово.
Программа выводит изменённую строку или сообщает, что такого слова нет.
После этого пользователь может выполнить замену ещё раз.

🧐Как запустить🧐  

Установите Python (если он ещё не установлен)  
Скачайте файл main.py  
Откройте терминал в папке с файлом  
Запустите программу командой:  
python main.py  
  
😇Как пользоваться😇  

 1.	Введите строку
	2.	Введите слово, которое хотите заменить
	3.	Введите новое слово
	4.	Получите результат на экране
	5.	Выберите, хотите ли выполнить ещё одну замену
  
👅Пример работы👅  

Ввод:  
Введите текст: я люблю программирование  
Какое слово заменить: программирование  
На что заменить: Python  
Вывод:  
Готово! Вот результат:  
я люблю Python  

Хотите выполнить ещё одну замену? (да/нет): нет  
Программа завершена.
